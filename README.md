# PyYTPLPlayer

**Requirements**
Python 3.8
```
pip3 install youtube-dl
pip3 install pyside2
```

**Shortcuts**
```
space - play/pause

# Volume
+ - volume up
- - volume down
m - mute/unmute

#Loop
[ - loop down
] - loop up
{ - loop zero
} - loop inf

# Seek
a - seek -1s
d - seek +1s
A - prev song
D - next song

# Playlists
, - highlight prev
. - highlight next
/ - select highlighted
b - add playlist from clipboard(copy link from yt)
B - remove highlighted

# Song list
l - highlight prev
; - highlight next
' - select highlighted
s - shuffle
```
